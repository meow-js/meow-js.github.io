# Meow.js
A way to communicate with the Scratch API.

NOTE: **All** functions are async.

## Install
This module is for [`Node.js`](https://nodejs.org/).

Install with npm:
```bash
npm install meowing
```
..or yarn:

```bash
yarn install meowing
```
